# Справочник услуг паспортного стола  
### Отдел по вопросам миграции Заволжского отдела полиции УМВД России по г. Твери

> Мобильное приложение для граждан, упрощающее подготовку к визиту в паспортный стол.  
> Учебный проект по практике — Android (Java), без интернета, полностью автономный.

---

## Назначение

Приложение помогает жителям г. Твери и Заволжского района:
- Подобрать нужную государственную услугу
- Узнать полный перечень необходимых документов
- Ознакомиться с режимом работы и контактами подразделения
- Подготовить «памятку» к визиту

**Важно**: приложение **не подаёт заявления** и не имеет юридической силы. Это справочный инструмент.

---

## Основные функции

- **Список услуг**  
  Выдача и замена паспорта, регистрация по месту жительства/пребывания, справки.
  
- **Чек-лист документов**  
  Для каждой услуги — список документов с возможностью отмечать готовые.

- **Информация об отделе**  
  Адрес, телефон, email, часы работы.  
  Кнопки: **«Позвонить»** и **«Показать на карте»** (с fallback в браузер).

- **Частые вопросы (FAQ)**  
  Раскрывающийся список типовых вопросов и ответов.

- **Памятка к визиту**  
  Рекомендации по подготовке (встроенная в описание услуг).
